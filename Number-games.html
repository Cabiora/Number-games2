Bienvenue sur Le Number-games

Bienvenue dans le jeu de nombres ! Devinez le bon nombre avec le moins de tentatives possible.

Le jeu vous demandera de deviner un nombre compris dans une plage donnée. À chaque essai, vous recevrez un indice pour savoir si le nombre que vous avez choisi est trop grand ou trop petit ou si ce nombre doit être paire ou impair. Essayez de deviner le bon nombre en un minimum d'essais
